# qc-v2

AI 终末病历质控系统原型。

## 在线访问

仓库配置完成后，可通过 GitHub Pages 或 Vercel 访问。

## 本地预览

```bash
python3 -m http.server 4173
```

然后打开：

```bash
http://127.0.0.1:4173/
```

## 当前原型能力

- 质控工作台总览
- 质检任务发起
- 多病例批量选择
- 批量导入入口（HIS/EMR、病例包、Excel 名单）
- 质检结果列表
- 单病例质控详情
- 质控报告页
- 整改复核
- 规则与看板

## 说明

当前为静态高保真原型，适合产品演示、流程评审和外部测试访问。
真实批量病例上传、后台解析、任务队列、报告生成等服务端能力后续可继续接入。
